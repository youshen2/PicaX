# PicaX

PicaX 是一个使用 SwiftUI 构建的多漫画源阅读客户端，面向 iOS、macOS 和 visionOS。项目以原生 Apple 平台体验为目标，把漫画源登录、浏览、搜索、收藏、阅读、下载、阅读统计、备份迁移和本地数据管理集中在一个应用里。

## 支持的漫画源

- PicACG
- JMComic
- NHentai
- E-Hentai / ExHentai
- Hitomi
- HT Manga

## 已有功能

### 账号与来源

- 多平台账号管理，支持 API 登录、网页登录、Cookie / User-Agent 保存与复用。
- 每个漫画源都有独立的前端地址配置，便于网页登录、分享链接解析和网页数据请求。
- PicACG 支持分流、图片质量、默认排序、收藏排序、头像框显示、资料页、打卡和我的评论。
- JMComic 支持 API 域名自动更新、图片分流、自定义图片地址、App 版本更新、收藏排序和签到。
- E-Hentai 支持 E-Hentai / ExHentai 切换、原图优先、内容警告跳过、标题偏好、Profile 选择、watched 订阅入口。
- Hitomi 支持图片数据域名配置。
- HT Manga 支持 API 地址和 API 分流更新。

### 浏览、搜索与发现

- 主页、收藏、发现、分类四个主标签页。
- 发现页支持按来源进入随机、最新和排行榜等入口。
- 分类页支持按来源加载分类，并跳转到对应搜索结果。
- 搜索页支持单平台搜索和多平台聚合搜索。
- 支持 PicACG、NHentai、JMComic 等来源的高级排序选项，NHentai 支持语言过滤。
- 搜索历史可记录关键词与来源，支持数量上限和清空。
- 支持剪贴板漫画链接检测、打开分享链接、JM 车牌号直达详情。
- 主页快捷工具包含 E-Hentai 订阅、搜图 bot 酱、SauceNAO、链接解析和 JM 车牌号入口。

### 详情、收藏与评论

- 漫画详情页展示封面、标题、作者 / 上传者、分类、标签、页数、热度、章节等信息。
- 支持本地收藏和各来源云端收藏夹。
- 支持收藏夹分页加载、完整加载后阅读全部。
- 支持从列表、标签、分类、收藏夹、历史记录和下载记录进入详情或阅读器。
- 支持评论区加载；支持的来源可发送评论或加载章节评论。
- 支持按标签继续浏览同类漫画。
- 列表可显示阅读进度、收藏状态、标签和热度，并可隐藏已读内容。
- 支持通用屏蔽词和 JMComic 专用屏蔽词。

### 阅读器

- 支持竖向连续滚动、竖向分页、左到右分页、右到左分页。
- 支持章节切换、章节列表 Sheet、阅读列表 Sheet、阅读进度和阅读历史保存。
- 支持批量阅读，把已下载漫画或收藏夹内漫画组成临时阅读列表，阅读器内可上一本 / 下一本切换，也可从阅读列表删除条目。
- 支持边界自动切换下一本，也可在设置里关闭。
- 支持点按翻页、自动翻页、两指缩放、双击缩放、长按临时缩放。
- 支持图片预加载、加载重试、深色模式降低图片亮度、图片间距、首图顶部留白和末图底部留白。
- 支持章节末尾显示评论，已下载漫画会优先使用本地保存的章节评论。
- 支持页码、进度浮层、时间电量浮层、液态玻璃背景、状态栏隐藏和 UI 显隐设置。
- 批量阅读切换书籍时可显示加载 Toast 和书名 Toast。

### 下载与本地阅读

- 支持按章节下载漫画，可选择下载全部或部分章节。
- 支持下载详情评论和章节评论。
- 支持下载队列、同时下载数、图片重试次数、下载限速。
- 下载页支持平台筛选、完成状态筛选、排序和删除。
- 已下载漫画可离线阅读，阅读列表只包含已下载章节。
- 已下载漫画支持阅读全部。
- 已下载漫画支持长按以 ZIP 导出，ZIP 内图片按阅读顺序命名，导出文件名格式可在设置里自定义，默认使用漫画标题。
- 首页可显示最近下载记录。

### 统计、历史与首页

- 阅读历史记录章节、页码和进度，支持清空历史或只清空阅读进度。
- 阅读时长统计支持今日、累计、单本详情和趋势图。
- 可设置阅读时长最多保存数量，以及单次阅读时长低于多少秒不记录。
- 首页卡片支持历史记录、阅读时长、下载、漫画源和快捷工具，并支持显示数量、折叠和排序设置。

### 数据、备份与迁移

- 支持图片缓存大小限制、空间占用查看和缓存清理。
- 支持导出 `.picax` 备份，可选择账号、设置、本地收藏、阅读历史、阅读时长、搜索历史、屏蔽词和已下载漫画。
- 支持导入 `.picax` 备份，导入前会显示预览，可选择覆盖或合并。
- 支持从 PicaComic 备份导入，覆盖账号、设置、下载、历史记录、本地收藏、搜索记录和屏蔽词等兼容数据。

## 相比原 PicaComic 更多的功能

相对原项目 `PicaComic` README 中列出的浏览、在线阅读、下载、本地 / 云端收藏、数据同步和阅读历史，PicaX 目前额外补充了：

- 原生 SwiftUI / Apple 平台 UI，以及 iOS 26+ Liquid Glass 适配。
- 阅读时长统计、单本趋势、首页阅读时长卡片和低于阈值不记录设置。
- 批量阅读：已下载、云端收藏夹、本地收藏夹等列表可直接阅读全部，并在阅读器内切换上一本 / 下一本。
- 搜索补全：NHentai平台的搜索补全
- 已下载漫画 ZIP 导出，并支持自定义 ZIP 文件名格式。
- PicaComic 备份迁移入口，导入前预览并支持覆盖 / 合并。
- 下载评论区、下载限速、下载高级筛选、离线阅读全部。
- 本地 `.picax` 精细化备份导出 / 导入，支持选择备份内容。

## 构建

1. 克隆仓库：

   ```bash
   git clone https://github.com/youshen2/PicaX.git
   cd PicaX
   ```

2. 使用 Xcode 打开项目：

   ```bash
   open PicaX.xcodeproj
   ```

3. 选择 `PicaX` scheme，并选择 iOS 模拟器、真机、macOS 或 visionOS 目标进行构建运行。

项目当前部署目标为 iOS 26.0 / macOS 26.0 / visionOS 26.0，Xcode 版本需能支持对应 SDK。
