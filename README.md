# PicaX

PicaX 是一个使用 SwiftUI 构建的多漫画源阅读客户端，支持漫画浏览、搜索、收藏、历史记录、阅读器、下载、本地缓存和备份等功能。

## 功能

- 多漫画源浏览、搜索和聚合搜索
- 漫画详情、章节阅读、阅读进度和历史记录
- 收藏、屏蔽词、列表阅读进度显示
- 漫画下载、下载队列和本地详情
- 图片缓存、存储管理和备份导入导出
- 阅读器自动翻页、点按翻页、缩放和深色模式图片亮度控制
- iOS 与 macOS 兼容适配

## 构建

1. 克隆仓库：

   ```bash
   git clone https://github.com/youshen2/PicaX.git
   cd PicaX
   ```

2. 使用 Xcode 打开项目：

   ```bash
   open PicaX.xcodeproj
   ```

3. 选择 `PicaX` scheme，并选择 iOS 模拟器、真机或 macOS 目标进行构建运行。
